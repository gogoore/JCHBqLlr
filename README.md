# 项目简介

## 核心功能
- 用户管理：用户注册登录、个人信息管理、会员充值、用户状态管理、用户禁用
- 场馆管理：场馆信息录入、场馆分类管理、场馆图片管理、场馆状态管理、场馆信息查询修改
- 预订管理：场地预订、预订审核、预订取消、预订统计、预订记录查询、待处理预订提醒
- 课程管理：课程发布、课程报名、课程评价、课程分类管理、课程状态管理
- 教练管理：教练信息管理、教练排班、教练评价、教练统计分析
- 财务管理：订单管理、收入统计、收入构成分析、日收入趋势、退款管理、财务报表
- 数据统计：用户统计、预订统计、收入统计、活跃用户分析、场馆利用率分析
- 管理后台：仪表板展示、最近订单查看、收入构成饼图、快捷操作入口、数据实时更新

## 适用场景
- 体育场馆运营管理
- 体育课程教学管理
- 用户活动组织与管理

## 技术特点
- 后端：使用 JDK17 和 SpringBoot3.2.7
- 前端：基于 Vue 3.3.10+Vite5.0.0 和 Element Plus
- 数据库：MySQL8.0+ MyBatis-Plus ORM
- 前后端分离架构，提高开发效率和系统可维护性
